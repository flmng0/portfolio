# Snake!

The classic game, featured for years on the Nokia handheld phones. See below for movement key options.

Eat as many apples as you can.

This is just a proof of concept for implementing things with ClojureScript, so there's nothing fancy or any win state.

Enjoy!

## Controls

- Arrow Keys, <kbd>w</kbd><kbd>a</kbd><kbd>s</kbd><kbd>d</kbd>, and <kbd>h</kbd><kbd>j</kbd><kbd>k</kbd><kbd>l</kbd> (Vim) can all be used to move.
- If you lose, you can press <kbd>r</kbd> to restart.
